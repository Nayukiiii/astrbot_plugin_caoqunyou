# astrbot_plugin_caoqunyou

AstrBot QQ 群插件——群友互草版本。不 @ 就随机挑一个群友，@ 指定人就草他！

> 此仓库由 [Claude](https://claude.ai) 编写

---

## 功能一览

| 功能 | 说明 |
|------|------|
| **草群友** | 随机 / 指定群友发起，密码学真随机判定成功率 |
| **草全群** | 一次性向全群所有人发起草草，每天限用 1 次，汇总显示成功/逃走/被反草数量 |
| **群友反草** | 草失败时目标有概率反草，仇恨值越高反草越猛 |
| **我的体内** | 查看自己（或 @ 某人）作为被草方的月度体内档案 |
| **我的战绩** | 查看自己（或 @ 某人）作为攻方的近 30 天战绩卡片 |
| **群友体内** | 全群体内液体排行（月度，被草维度） |
| **群友战绩** | 全群互草战绩排行（草人注入总量榜 + 被草次数榜） |
| **杂鱼排行** | 近 30 天选外面的杂鱼排行 |
| **草群友排行** | 主动草人次数榜 |
| **草群友关系图** | 可视化互草关系有向网络图（纯 Pillow 渲染，KK 布局，全量边透明度叠加） |
| **草限定** | 每日随机抽取限定群友，每次逃走后成功概率递增 |
| **重置草群友** | 重置今日草群友次数（普通用户需掷骰，管理员直接重置） |
| **重置草全群** | 重置今日草全群冷却（普通用户需掷骰，管理员直接重置） |

## 草全群机制

- 每人每天限用 **1 次**（独立冷却，不消耗每日草群友次数）
- 对群内所有成员逐一以相同概率判定，**自动排除 bot 自身**
- 成功者默认 **射里面**（记录体内数据和战绩），静默处理不弹选项
- 反草逻辑**保留**：若被反草，同样默认里面静默结算，并计入关系图
- 最终发送一条汇总消息：`✅ 成功 X 人 / 🏃 逃走 X 人 / ⚔️ 被反草 X 人`

## 草群友关系图

使用 **Pillow 纯 Python 渲染**，不依赖 Playwright / vis-network，速度快稳定性高。

- 布局算法：Kamada-Kawai 近似（Floyd-Warshall + 梯度下降能量最小化）
- 边渲染：全量显示，RGBA 透明度叠加——草 1 次淡，草多次深，密集区自然形成热力效果
- 节点大小：按草出次数加权，草得越多头像越大
- 箭头：实心三角形，互草（双向）用紫色偏移避免重叠，单向用玫红

## 配置项

| 字段 | 说明 | 默认值 |
|------|------|--------|
| `cao_probability` | 草群友触发概率（%） | 30 |
| `fancao_probability` | 反草基础强度（%），设 0 关闭反草 | 50 |
| `cao_daily_limit` | 每人每天草群友次数上限 | 5 |
| `reset_daily_attempts` | 普通用户每天重置尝试次数上限 | 3 |
| `allow_self_cao` | 是否允许自草 | false |
| `whitelist_groups` | 功能白名单群（留空=全部群可用） | [] |
| `blacklist_groups` | 功能黑名单群 | [] |
| `announce_groups` | 每日限定推送群列表 | [] |
| `announce_hour` | 每日限定推送时间（小时，24h制） | 12 |
| `xd_base_probability` | 草限定基础概率（-1 表示复用草群友概率） | -1 |
| `xd_escape_bonus` | 草限定逃走后概率加成（%/次） | 5 |
| `xd_max_probability` | 草限定概率上限（%） | 95 |
| `auto_withdraw_enabled` | 开启定时自动撤回（仅 aiocqhttp） | false |
| `auto_withdraw_delay_seconds` | 自动撤回延迟（秒） | 5 |

## 概率机制

### 草群友触发

使用 `secrets` 模块（操作系统熵源）生成真随机数，不可预测、不可复现。

### 注入量

对数正态双峰分布，日常集中在几十到几百 mL，上限 100L。

| 评级 | 范围 |
|------|------|
| 雕王 | ≥ 4000 mL |
| 半雕王 | 2000–3999 mL |
| 半死不活 | 800–1999 mL |
| 杂鱼～ | < 800 mL |

### 反草触发

草群友失败后，目标反草概率由四个因子动态计算：

- **Q — 逃脱质量**：用 sigmoid 函数将逃走百分比转换，越险逃反草概率越高
- **G — 累积仇恨**：该用户近 30 天被草次数越多越记仇
- **T — 今日压力**：当天群内已被草次数越多，群友反扑意志越强
- **E — 随机熵扰动**：每次判定注入真随机扰动（±6%）

## 数据说明

- **群友体内数据**：每月 1 日 0 点自动重置
- **战绩 / 反草 / 杂鱼数据**：滚动统计近 30 天
- **草全群冷却**：按自然日（每天 0 点重置）
- 所有数据存储于 AstrBot `data` 目录，插件更新不会丢失
